# AI Forecast & Tech News Assistant

Conversational AI assistant built with n8n, OpenAI, Gemini and external APIs.  
The project combines real-time weather forecasting and technology news retrieval while exploring LLM orchestration, tool calling, contextual memory and conversational interfaces.

---

## Overview

This project explores the development of AI-powered workflows using n8n and modern language models.

The assistant can:
- provide weather forecasts through natural language,
- retrieve recent technology news,
- interpret relative dates such as “tomorrow” or “this weekend”,
- maintain short-term conversational context,
- and interact with external APIs through tool calling.

---

## Tech Stack

- n8n
- OpenAI API
- Google Gemini API
- Open-Meteo API
- RSS Feed integrations
- Render

---

## Features

- Conversational weather forecasting
- Technology news retrieval
- Short-term conversational memory
- Relative date interpretation
- Real-time API integration
- AI tool orchestration

---

## Workflow Architecture

The workflow consists of:

1. Chat interface and trigger
2. AI agent orchestration
3. Conversational memory
4. Weather API integration
5. RSS-based tech news retrieval
6. OpenAI and Gemini language models
